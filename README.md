# HoloPhysics Lab 2 — Научная физическая песочница

**Нативное Windows-приложение на C++20 / OpenGL 4.6**

Высокоточная научная симуляция физических процессов с профессиональным интерфейсом.

---

## 🚀 Возможности

### Физическое ядро
- **Double precision** для всех вычислений
- **6 численных методов**: Явный Эйлер, Симплектический Эйлер, Velocity Verlet, RK4, RK5 (Фельберг), Рут (3-й порядок, симплектический)
- **20+ физических сил**: гравитация, ветер, сопротивление, Архимед, магнитное поле, Кулон, вихрь, Кориолис, турбулентность, когезия, пружины, центральная гравитация
- **Контроль сохранения энергии** — автоматическое отслеживание дрейфа
- **До 1000+ тел** в реальном времени

### Рендеринг
- **OpenGL 4.6** Core Profile
- 4 формы тел: сфера, куб, цилиндр, тор
- Научная визуализация с минималистичным дизайном
- Траектории, коллизионные маркеры, сетка

### Интерфейс
- **Строгий научный UI** в стиле MATLAB/Origin
- **Dock-панели** с произвольной компоновкой
- **ImPlot** — профессиональные научные графики
- Графики энергии (E_k, E_p, E_total), импульса, дрейфа

### Управление сценами
- Сохранение/загрузка в JSON (слоты 1-10)
- Drag & Drop JSON-файлов
- Горячие клавиши для быстрых действий
- 9 встроенных пресетов

## 🛠 Сборка

### Требования
- **Windows 10/11** (или Linux с малыми изменениями)
- **Visual Studio 2022** с MSVC v143
- **CMake** 3.20+
- **vcpkg** (для управления зависимостями)

### Шаг 1: Установка vcpkg
```powershell
git clone https://github.com/Microsoft/vcpkg.git
cd vcpkg
.\bootstrap-vcpkg.bat
.\vcpkg integrate install
```

### Шаг 2: Установка зависимостей
```powershell
cd vcpkg
.\vcpkg install glfw3 glad glm imgui implot nlohmann-json --triplet x64-windows
```

Или используйте manifest mode (автоматически):
```powershell
cd HoloPhysicsLab2
cmake --preset default
```

### Шаг 3: Сборка
```powershell
mkdir build && cd build
cmake .. -G "Visual Studio 17 2022" -DCMAKE_BUILD_TYPE=Release -DCMAKE_TOOLCHAIN_FILE="путь\к\vcpkg\scripts\buildsystems\vcpkg.cmake"
cmake --build . --config Release
```

### Шаг 4: Запуск
```powershell
.\Release\HoloPhysicsLab2.exe
```

## 🧪 Физические методы

| Метод | Порядок | Сохраняет энергию | Применение |
|-------|---------|-------------------|------------|
| Явный Эйлер | 1 | Нет | Демонстрация, сравнение |
| Симплектический Эйлер | 1 | Да | Быстрые симуляции |
| Velocity Verlet | 2 | Да | Молекулярная динамика |
| RK4 | 4 | Нет | Высокая точность на шаг |
| RK5 (Фельберг) | 5 | Нет | Максимальная точность |
| Рут | 3 | Да | Долгие симуляции |

### Контроль сохранения энергии
Автоматически вычисляется относительный дрейф энергии:
```
Δ = |E(t) - E(0)| / |E(0)|
```
Отображается в реальном времени и на графике. Позволяет выбрать оптимальный интегратор.
